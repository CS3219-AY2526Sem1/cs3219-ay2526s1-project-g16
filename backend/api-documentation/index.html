Subscribe to matchmaking updates via SSE

Opens a Server-Sent Events (SSE) stream for real-time matchmaking updates. The stream sends the following events:

  • heartbeat: every 15s
  • MATCH_FOUND: when a match is found
  • TIMEOUT: if no match found before ttlMs expires
Authorizations:
jwtTokensCookieAuth
path Parameters
userId
required
string
Example: user123
query Parameters
ttlMs
integer
Example: ttlMs=30000
pollMs
integer
Example: pollMs=1000

Responses

PeerPrep API

https://localhost:8080/matching/subscribe/{userId}

  • 200
  • 400
